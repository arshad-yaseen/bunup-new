# my-react-lib

A react component library

## Installation

```bash
npm install my-react-lib
```

## Usage

```typescript
import { greet } from 'my-react-lib';

console.log(greet('World')); // Hello, World!
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Develop with watch mode
bun run dev

# Run tests
bun run test
```
