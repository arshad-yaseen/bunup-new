# bunup-new

Quickly scaffold new modern projects with Bun

## Installation

```bash
npm install bunup-new
```

## Usage

```typescript
import { greet } from 'bunup-new';

console.log(greet('World')); // Hello, World!
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Develop with watch mode
bun run dev

# Run tests
bun run test
```
