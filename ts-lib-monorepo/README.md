# ts-lib-monorepo

A typescript library

## Installation

```bash
npm install ts-lib-monorepo
```

## Usage

```typescript
import { greet } from 'ts-lib-monorepo';

console.log(greet('World')); // Hello, World!
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Develop with watch mode
bun run dev

# Run tests
bun run test
```
